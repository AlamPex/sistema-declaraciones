\# 📋 Sistema de Declaraciones



\## 📁 Estructura del Proyecto

\- \*\*backend/\*\* - API y lógica del servidor

\- \*\*frontend/\*\* - Interfaz de usuario

\- \*\*reportes/\*\* - Generación de reportes



\## 🚀 Instalación



\### Backend

cd backend

npm install

npm start



\### Frontend

cd frontend

npm install

npm start



\### Reportes

cd reportes

npm install

npm start
